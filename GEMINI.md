## 빠른 요약

버추얼 스트리머 챠이로 쿠냐의 비공식 팬 사이트. **정적 HTML/CSS/JS · 빌드 없음 ·
의존성 없음 · 테스트 없음.** 저장소 루트를 GitHub Pages 가 그대로 서빙한다
(<https://chnu-kim.github.io/chyaro-kunya/>).

```bash
python3 -m http.server    # 로컬 서버 → http://localhost:8000 (file:// 로 열지 말 것)
```

빌드도 린트도 테스트도 없으므로 잘못된 변경을 잡아줄 자동 장치가 없다. 브라우저에서
직접 확인하거나 배포 URL 을 찔러 확인한다.

## 작업 전 반드시

1. **`AGENTS.md` 를 끝까지 읽는다.** 특히 "이 코드베이스에서 실제로 밟은 지뢰" 절 —
   SVG 기본 fill, 한글 폰트 폴백, 인터랙티브 요소의 role, og:image 절대 URL 은
   전부 여기서 실제로 터진 버그다.
2. **디자인 토큰을 확인한다.** `css/site.css` 의 `:root` 가 색·타입의 정본이다.
   생 hex 를 새로 쓰지 않고 `var(--token)` 으로만 참조한다.
3. **채널은 치지직·유튜브·X 3개뿐이다.** 디스코드는 없다 — 만들지 않는다.

## Gemini 관련

- 사용자 노출 문구는 전부 **한국어**로 쓴다. 코드 식별자·브랜드명은 원문 유지.
- 이 저장소는 **공개**다. 커밋 전 `git status` 로 스테이징을 확인한다.
  `.gitignore` 가 스크래치 이미지와 내부 메타데이터를 막고 있다 — 우회하지 않는다.
- 사이트에 넣을 이미지를 **생성하지 않는다.** 필요한 목록을 정리해 사용자에게 요청한다.
  반대로 장식 요소(발바닥·고양이·리본)는 인라인 SVG 로 직접 작성한다.
- 파일을 고치기 전에 읽는다. nav·푸터 마크업은 세 페이지에 복제돼 있어서
  한 곳만 고치면 나머지 둘과 어긋난다.
